# 界面测试用例：歌单管理功能

## 功能概述

歌单管理界面包括：歌单列表展示、创建歌单、删除歌单、重命名歌单、歌单详情页、编辑模式批量操作。

## 测试范围

- 测试层级：界面测试
- 测试对象：MyMusicScreen 的歌单 Tab、PlaylistScreen
