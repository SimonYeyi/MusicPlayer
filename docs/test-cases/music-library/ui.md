# 界面测试用例：音乐库功能

## 功能概述

音乐库页面（MusicLibraryScreen）展示本地音乐的分类入口界面，包括 TopAppBar 标题栏和 6 个分类列表项。

## 测试范围

- 测试层级：界面测试
- 测试对象：MusicLibraryScreen、LibraryCategoryItem

---
