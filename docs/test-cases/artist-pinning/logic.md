# 逻辑测试用例：歌手置顶功能

## 功能概述

歌手置顶功能包括：长按歌手名称置顶/取消置顶，置顶歌手优先显示在列表顶部。

## 测试范围

- 测试层级：单元测试
- 测试对象：MusicViewModel 的置顶歌手相关逻辑
