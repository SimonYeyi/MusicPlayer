# 界面测试用例：歌手置顶功能

## 功能概述

歌手置顶界面包括：歌手分组显示、长按歌手名称置顶/取消置顶操作。

## 测试范围

- 测试层级：界面测试
- 测试对象：MyMusicScreen 的本地音乐 Tab

---

## 测试用例统计

| 正常交互 | 边界交互 | 异常交互 | 总计 |
|---------|---------|---------|------|
| 7 | 3 | 2 | 12 |

---

## 测试用例

### TC-PIN-UI-001: 本地音乐 Tab - 按歌手分组显示

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-PIN-UI-001 |
| 测试点描述 | 本地音乐列表应按歌手分组显示 |
| 输入 | 有歌曲：周杰伦[A,B]、林俊杰[C]、王菲[D,E,F] |
| 预期输出 | 分组显示：周杰伦、林俊杰、王菲 |
| 测试类型 | 正常交互 |
