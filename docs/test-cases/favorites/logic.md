# 逻辑测试用例：收藏管理功能

## 功能概述

收藏管理功能包括：收藏/取消收藏歌曲、收藏列表展示。

## 测试范围

- 测试层级：单元测试
- 测试对象：MusicRepository 的收藏相关方法、MusicViewModel 的收藏逻辑
