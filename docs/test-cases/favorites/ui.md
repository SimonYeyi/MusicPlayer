# 界面测试用例：收藏管理功能

## 功能概述

收藏管理界面包括：收藏列表展示、收藏/取消收藏歌曲操作。

## 测试范围

- 测试层级：界面测试
- 测试对象：MyMusicScreen 的收藏 Tab
