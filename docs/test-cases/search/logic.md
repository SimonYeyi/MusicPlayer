# 逻辑测试用例：搜索功能

## 功能概述

搜索功能：在本地音乐中搜索歌曲，支持按歌曲标题和艺术家名称搜索。

## 测试范围

- 测试层级：单元测试
- 测试对象：MusicViewModel 的搜索逻辑、MusicRepository 的搜索方法
