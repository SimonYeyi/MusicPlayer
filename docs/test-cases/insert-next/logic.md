# 插入下一首播放逻辑测试用例

## 功能概述

插入下一首播放功能允许用户将歌曲插入到当前播放位置的下一首位置。支持顺序播放和随机播放两种模式，并处理暂停状态的特殊逻辑。

## 测试用例统计

| 正常流程 | 边界条件 | 异常处理 | 总计 |

| 9 | 5 | 2 | 16 |

## 测试用例

### TC-001: 顺序播放模式 - 插入不在队列中的歌曲

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-001 |
| 测试点描述 | 顺序播放模式下，将不在当前播放队列中的歌曲插入到下一首位置 |
| 输入 | 播放列表 [A, B, C, D]，currentIndex=0（播放A），插入歌曲X |
| 预期输出 | 队列变为 [A, X, B, C, D]，currentIndex 保持为 0 |
| 测试类型 | 正常流程 |

---

### TC-002: 顺序播放模式 - 插入已在队列中的歌曲（位于插入点）

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-002 |
| 测试点描述 | 顺序播放模式下，将已在队列中且恰好位于插入点（当前歌曲之后）的歌曲 |
| 输入 | 播放列表 [A, B, C, D]，currentIndex=0（播放A），插入歌曲B（index=1，即插入点位置） |
| 预期输出 | 队列变为 [A, B, C, D]（B 已在插入点，无变化），currentIndex 保持为 0 |
| 测试类型 | 边界条件 |

---
