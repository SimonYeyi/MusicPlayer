# 插入下一首播放界面测试用例

## 功能概述

插入下一首播放功能通过歌曲列表的长按菜单或更多按钮菜单触发，插入成功后显示 toast 提示。

## 测试用例统计

| 正常交互 | 边界交互 | 异常交互 | 总计 |
|---------|---------|---------|------|
| 6 | 2 | 2 | 10 |

---

## 测试用例

### TC-UI-001: 歌曲列表 - 长按显示菜单

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-UI-001 |
| 测试点描述 | 长按歌曲应显示操作菜单，包含"下一首播放"选项 |
| 输入 | 在本地音乐列表，长按歌曲B |
| 预期输出 | 显示 BottomSheet 菜单，包含"下一首播放"选项 |
| 测试类型 | 正常交互 |

---

### TC-UI-002: 歌曲列表 - 点击更多按钮显示菜单

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-UI-002 |
| 测试点描述 | 点击歌曲行的更多按钮应显示操作菜单 |
| 输入 | 在歌曲列表，点击歌曲B的更多按钮（⋮） |
| 预期输出 | 显示 DropdownMenu 或 BottomSheet，包含"下一首播放"选项 |
| 测试类型 | 正常交互 |

---

| 字段 | 值 |

### TC-UI-004: 插入当前歌曲时不显示提示

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-UI-004 |
| 测试点描述 | 当插入的歌曲是当前正在播放的歌曲时，不显示提示也不触发插入 |
