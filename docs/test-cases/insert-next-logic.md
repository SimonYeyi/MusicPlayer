# 逻辑测试用例：插入下一首播放功能

## 功能概述

在播放音乐时，允许用户将歌曲插入到当前播放位置的下一首位置。支持顺序播放和随机播放两种模式，并处理暂停状态的特殊逻辑。

## 测试范围

- 测试层级：单元测试
- 测试对象：MusicPlaybackService 的 insertNext 相关业务逻辑

---

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-001 |
| 测试点描述 | 顺序播放模式下，将不在当前播放队列中的歌曲插入到下一首位置 |
| 输入 | 播放列表 [A, B, C, D]，currentIndex=0（播放A），插入歌曲X |
