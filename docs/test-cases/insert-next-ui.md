# 界面测试用例：插入下一首播放功能

## 功能概述

在播放音乐时，允许用户通过长按歌曲或点击菜单，将歌曲插入到当前播放位置的下一首位置。支持从"我的音乐"页面、"歌单"页面等入口触发插入操作。

## 测试范围

- 测试层级：界面测试
- 测试对象：MyMusicScreen、PlaylistScreen、HomeScreen 中的插入下一首交互

---

## 测试用例

### TC-UI-001: 我的音乐页面 - 长按歌曲显示操作菜单

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-UI-001 |
| 测试点描述 | 在我的音乐页面长按非当前播放的歌曲，应显示包含"插入下一首"的操作菜单 |
| 输入 | 我的音乐页面，播放A，长按歌曲B |
| 预期输出 | 显示底部操作菜单，包含"插入下一首"选项 |
| 测试类型 | 正常交互 |
