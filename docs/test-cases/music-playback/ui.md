# 界面测试用例：音乐播放功能

## 功能概述

音乐播放界面功能包括：播放详情页显示专辑封面（旋转动画）、歌曲名、艺术家、进度条控制、播放控制按钮、播放队列（底部弹窗）。

## 测试范围

- 测试层级：界面测试
- 测试对象：PlayDetailScreen、MiniPlayer、HomeScreen
