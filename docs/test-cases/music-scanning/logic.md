### TC-MUSIC-SCAN-001

| 字段 | 说明 |
|------|------|
| 测试用例编号 | TC-MUSIC-SCAN-001 |
| 测试点描述 | 扫描本地音乐文件，验证正常扫描流程 |
| 输入 | 设备存储中有3首MP3音频文件 |
| 预期输出 | 成功获取3首歌曲信息，包含正确的标题、艺术家、专辑、时长 |
| 测试类型 | 正常流程 |
