# 设置功能逻辑测试用例

## 功能概述

设置功能包括：主题设置、播放模式切换、音乐管理（本地音乐数量显示、重新扫描）、播放历史入口、版本信息、关于信息。

## 测试用例统计

| 正常流程 | 边界条件 | 异常处理 | 总计 |
|---------|---------|---------|------|
| 6 | 2 | 2 | 10 |

## 测试用例

### TC-SET-001: 主题设置 - 切换心情主题

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-SET-001 |
| 测试点描述 | 用户选择心情主题后应正确保存并应用 |
| 输入 | 选择 MoodTheme.CALM |
| 预期输出 | 主题保存到 SharedPreferences，UI 刷新为 CALM 主题 |
| 测试类型 | 正常流程 |

---

### TC-SET-002: 主题设置 - 主题持久化

| 字段 | 值 |
|------|-----|
| 测试用例编号 | TC-SET-002 |

| 输入 | 已选择 HAPPY 主题，杀死进程后重启 |
