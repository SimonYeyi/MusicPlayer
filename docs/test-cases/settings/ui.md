# 设置功能界面测试用例

## 功能概述

设置界面包括：返回按钮、心情主题选择入口、播放模式切换、本地音乐数量显示、重新扫描入口、跳转收藏/最近播放、版本信息、关于信息。

| 10 | 2 | 2 | 14 |
