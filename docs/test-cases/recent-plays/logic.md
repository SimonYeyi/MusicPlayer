# 逻辑测试用例：最近播放功能

## 功能概述

最近播放功能包括：自动记录播放历史、清除播放历史、从最近播放移除歌曲。

## 测试范围

- 测试层级：单元测试
- 测试对象：MusicRepository 的最近播放相关方法、MusicViewModel 的最近播放逻辑
