# 界面测试用例：最近播放功能

## 功能概述

最近播放界面包括：最近播放列表展示、播放历史清除、从最近播放移除歌曲。

## 测试范围

- 测试层级：界面测试
- 测试对象：MyMusicScreen 的最近播放 Tab
