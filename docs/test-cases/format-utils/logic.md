# 逻辑测试用例：格式工具类

## 功能概述

FormatUtils 工具类提供时间格式化功能，将毫秒时长格式化为可读的分:秒或时:分:秒格式。

## 测试范围

- 测试层级：单元测试
- 测试对象：formatDuration 函数
