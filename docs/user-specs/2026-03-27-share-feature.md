# 分享功能 - 用户视角

## 功能描述

用户可以在歌曲列表中长按任意歌曲，或在播放详情页点击分享按钮，将歌曲分享给其他应用。

分享的内容：
- 文本信息："分享歌曲：{歌曲名} - {艺术家}"
- 音频文件本身

## 用户场景

| 场景 | 用户操作 | 系统响应 |
|------|----------|----------|
| 分享歌曲 | 长按歌曲 → 点击"分享" | 弹出系统分享面板 |
| 分享给好友 | 在分享面板选择微信/QQ等 | 发送歌曲信息和音频文件 |
| 分享失败 | 点击分享但无可处理应用 | 显示 Toast 提示 |
| 分享失败 | 点击分享但文件无效 | 显示 Toast 提示 |

## 界面位置

- **歌曲列表长按菜单**：位于"收藏"下方、"设为铃声"上方
- **播放详情页更多菜单**：与"添加到歌单"、"收藏"、"设为铃声"并列
- **播放详情页 TopAppBar**：右侧，与返回按钮并列
